Sélectionnez le bon chemin relatif entre le fichier "index.html" et le fichier "mon-image.gif"

Rappel sur le chemin relatif :

  • Lien pointant vers une ressource dont l'emplacement est relatif au fichier qui l'appelle
  • "./" indique au navigateur qu’il doit chercher l’image à partir du dossier courant
  • Si les fichiers ne sont pas dans le même dossier, le caractère slash (/) sert de séparateurs entre les dossiers
  • Pour remonter d'un niveau dans l'arborescence il faut mettre "../" dans le chemin du fichier que l'on souhaite atteindre.
    A noter que "./" et "../" dans le chemin d'un fichier est possible mais aberrant
