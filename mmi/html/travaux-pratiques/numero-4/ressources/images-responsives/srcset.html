Image responsive - Partie 1

Consigne : Afficher une image d'une grande largeur (> 1400px)
Redimensionnez votre fenêtre. L'image devrait "dépasser" de la fenêtre et créer un défilement horizontal.
Si vous n'avez pas d'images il y a une image assets/images/panorama.jpg

Image responsive - Partie 2

Consigne : Ajouter une classe "img-responsive" dans un fichier css avec les déclarations suivantes : max-width: 100 %; height: auto;
Ajouter la classe sur l'image précédemment mise ou vous pouvez mettre en-dessous une image avec la classe que vous avez ajouté
