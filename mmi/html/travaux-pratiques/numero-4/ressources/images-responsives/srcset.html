Travaux pratiques sur les images responsives

But du TP : Voir les attributs srcset et sizes

Image responsive - Partie 1

Consigne : Afficher une image d'une grande largeur (> 1400px)
Redimensionnez votre fenêtre. L'image devrait "dépasser" de la fenêtre et créer un défilement horizontal.
Si vous n'avez pas d'images il y a une image assets/images/panorama.jpg

Image responsive - Partie 2

Consigne : Ajouter une classe "img-responsive" dans un fichier css avec les déclarations suivantes : max-width: 100 %; height: auto;
Ajouter la classe sur l'image précédemment mise ou vous pouvez mettre en-dessous une image avec la classe que vous avez ajouté

Attributs srcset et sizes - Partie 1

Consignes

  • Ajouter une balise <img /> avec les attributs srcset et src
  • Mettre comme valeur pour l'attribut src, une image du dossier "assets/images/attribut-srcset/densite"
  • Définir la valeur de l'attribut srcset avec autant de valeurs qu'il y a d'images dans le dossier "assets/images/attribut-srcset/densite" avec le descripteur de densité (x) jusqu'à 3x
  • Tester avec le mode responsive du navigateur (ctrl + maj + m sur Firefox et Chrome / Clic droit -> Inspecter -> cliquer sur le smartphone à gauche de l'onglet "éléments" de la console du navigateur)
    Jouer sur la taille de l'écran

Attributs srcset et sizes - Partie 2

Consignes

  • Refaire la même chose avec des descripteurs de largeur (w) avec comme valeur le nombre contenu dans le nom de l'image. Les images contenues dans le dossier "assets/images/attribut-srcset/largeur"
  • Ajouter l'attribut sizes avec des media queries et une taille
    Exemple : Ex : (max-width : 480px) 100vw, (min-width : 481px) 50vw
    Ne pas oublier la valeur par défaut
  • Tester avec le mode responsive du navigateur (ctrl + maj + m sur Firefox et Chrome / Clic droit -> Inspecter -> cliquer sur le smartphone à gauche de l'onglet "éléments" de la console du navigateur)
    Jouer sur la taille de l'écran
